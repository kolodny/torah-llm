# 017 — Jewish-origin sources only; + Open Siddur liturgy

**Date:** 2026-06-09

## Policy: Jewish-origin sources only

Per the maintainer, the project ingests **Jewish-origin sources only**. Consequences:

- **Removed OpenScriptures `morphhb` (WLC), former source #2.** It's the Masoretic Hebrew *text*
  (Jewish), but the digitizing project is Christian-affiliated. Pulled the adapter, its lock entry,
  and cached data; rebuilt. Net: Genesis 6→5 editions; **Jonah/Ruth dropped** (they were WLC-only).
  Sefaria still supplies the Hebrew Tanakh.
- **Excluded from the [014](014.data-sources.md) recommendations:** STEPBible (Tyndale House),
  OpenScriptures HebrewLexicon, tanach.us/UXLC (Westminster Theological Seminary lineage) — all
  Christian-affiliated. (We never ingested these — recommendations only.)
- The Jewish-origin path for morphology/nikud is **Dicta** (Israeli), not STEPBible/OpenScriptures.
- Current sources: **Sefaria + Orayta + Open Siddur** (all Jewish orgs). Sefaria editions kept
  include JPS 1917 (Jewish Publication Society) and Zadoc Kahn (French Grand Rabbi) — Jewish-origin.

## Added: Open Siddur liturgy

`ingest/sources/opensiddur.ts` — from `github.com/opensiddur/sourcetexts` (pinned). Open Siddur's
git content: the clean **CC0 JLPTEI** is example-sized, while the substantial **whole-siddurim** are
flat `.txt`/`.fodt`/`.html`. We ingest the **plain-text, clearly public-domain** ones under the
canonical **`Liturgy`** category (which already exists in Sefaria's TOC spine):

- **Seder ha-Tefillot (Rambam)** — `.txt`, `[gematria]` section markers → `section:paragraph` refs.
  167 rows, voweled Hebrew. PD (Maimonides, 12th c.).
- **Standard Prayer Book (Singer)** — `.txt`, `{file "…"}` section markers (Title Page / Table of
  Contents skipped; `{rem …}`/`{…}`/`<…>` stripped; `�`→`'`). 3152 rows, English. PD (Bloch, 1915).

This is the first adapter to **create its own canonical TOC nodes** (a book under `Liturgy`) rather
than attach to a Sefaria-defined book. Verified in-browser: the Rambam siddur loads (its slice
filename has spaces + parens — exercises the double-encoded path) and renders voweled Hebrew RTL with
its provenance tooltip.

**Deferred:** Fanny Neuda *Stunden der Andacht* (`.fodt`, German, PD) and Siddur Tehillat haShem
(`.fodt` ×2 — base PD but modern Chabad layout may be copyrighted) need ODF parsing + license
vetting; "Hanna" (`.html`) is unidentified. The fuller structured corpus is behind Open Siddur's
live eXist API (per-text TEI-header licenses) — a non-pinnable dependency, deferred.
