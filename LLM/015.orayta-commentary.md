# 015 — Orayta commentary ingest (+ where Orayta's links actually live)

**Date:** 2026-06-09

## Correction: Orayta *does* carry cross-references — in the per-book `.conf`

An earlier assumption (that Orayta has no link data) was wrong — it came from grepping only the
base-text `.txt`. Each base book's **`.conf`** lists its attached commentaries via `AddSource`
lines. `a01_Genesis.conf`:

```
AddSource 005_mprsi_mkra/03_rsi/…RASHI_BERESHIT…:פירוש רש"י
AddSource 005_mprsi_mkra/04_rmbn/…:פירוש הרמב"ן
AddSource 005_mprsi_mkra/05_abn_uzra/…:פירוש אבן עזרא
# + Onkelos, Targum Yonatan, Baal HaTurim, Or HaChaim, Sforno, Siftei Chachamim, Keli Yakar…
```

These are **book-level** associations (like Sefaria's TOC `dependence: Commentary` /
`base_text_titles`), not verse↔verse pairs. Verse-level links come from ingesting the commentary
*text* (Orayta aligns commentary to verse by parallel structure). Commentary texts live under
`BooksSrc/005_mprsi_mkra/…`, Midrash under `010_mdrs`, Talmud under `030_tlmod_bbli`, etc.

## Ingested: Rashi on the Torah (a 3rd edition overlaying Sefaria)

`ingest/sources/orayta.ts` now also ingests the 5 Rashi-on-Torah `.txt` files as Hebrew editions of
the **existing canonical** "Rashi on Genesis" … "Rashi on Deuteronomy" books.

- **Comment splitting:** each `! {verse}` block holds all of that verse's Rashi; individual comments
  are delimited by `<b>catchphrase</b>` (dibur hamatchil). Split on `<b>` → ref `chapter:verse:comment`.
- **Gematria refs:** commentary verse markers skip verses with no comment, so counting is unsafe —
  the chapter (`~ … פרק-<gem>`) and verse (`! {<gem>}`) labels are gematria, converted to integers.
- **Alignment verified:** Sefaria and Orayta produce *identical* comment counts per verse (Gen ch.1:
  3,5,1,1,2,2,1… both skip 1:3), so Orayta's `c:v:comment` refs line up with Sefaria's and the
  editions overlay row-for-row. Confirmed in-browser: 3 columns (Sefaria he + en + Orayta he) show
  the same comment per row at `1:1:1`, `1:1:2`, `1:1:3`. Row counts: Orayta 2006 vs Sefaria 2017
  on Rashi-Genesis (~99.5%).
- **Cleaning:** strip the editor's `<small>…</small>` source-notes and `{{…}}` footnote anchors,
  keep the (de-bolded) catchphrase, tidy whitespace.
