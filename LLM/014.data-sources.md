# 014 · Open data sources to ingest (research)

Generated by a research subagent on **2026-06-09**, licenses/formats verified against live pages
+ the GitHub API (not from memory). Currently ingested: Sefaria-Export (GCS), OpenScriptures
morphhb/WLC, Orayta. This catalogs sources to add *beyond* those. See [0.index](./0.index.md).

> **Policy update ([017](017.jewish-sources-and-liturgy.md)):** the project ingests **Jewish-origin
> sources only**. This **excludes** the STEPBible, OpenScriptures (incl. HebrewLexicon), and
> tanach.us/UXLC picks below (Christian-affiliated projects). The Jewish-origin picks still apply:
> **Dicta** (Talmud-with-Nikud, Library, Nakdan), **Open Siddur** (✓ added), **PocketTorah**,
> **Hebrew Wikisource**. For morphology/nikud, the Jewish-origin path is **Dicta**, not STEPBible.

current three sources have.

## HIGH value

- **STEPBible-Data** — github.com/STEPBible/STEPBible-Data. TAHOT (tagged Hebrew OT, morphology +
  Strong's, scan-corrected), TAGNT (Greek NT), TBESH/TBESG/TFLSJ lexicons, TIPNR names, TVTMS
  versification. Format: UTF-8 TSV `.txt` (some `$`-delimited multiline). License **CC BY 4.0** in the
  filenames (skip `TTESV` = CC BY-NC). Pin by commit SHA. Beats morphhb (adds semantic tags, scan
  corrections, Greek + lexicons).
- **Dicta Talmud-Bavli-with-Nikud** — github.com/Dicta-Israel-Center-for-Text-Analysis/Talmud-Bavli-with-Nikud.
  Whole Shas, vocalized, one dir/tractate, `…haser.txt` + `…maleh.txt`. **CC BY-SA** (README). Small
  (~10 MB). Fills the biggest gap: a fully-pointed Bavli.
- **tanach.us (UXLC)** — full Tanakh, nikud + te'amim, validated XML (`Tanach.xsd`), Ketiv/Qere. Text
  **"may be viewed or copied without restriction"** (fonts GPL/separate). **Publishes SHA-256 + build
  number per release** → gold-standard pin even though the URL is mutable. ~2.3 MB. Same text family as
  morphhb (duplicative on text, superior for reproducibility + clean reading text).
- **OpenScriptures HebrewLexicon** — BDB + Strong's Hebrew + LexicalIndex (BDB↔Strong's↔TWOT), XML +
  XSD. **CC BY 4.0**. Small. New layer (tap-a-word definitions, keyed to Strong's like morphhb/TAHOT).
  Note: HALOT is copyrighted Brill — not available, don't chase it.
- **Dicta Library-Download** — ~300 rabbinic books (responsa, commentaries) as TXT, `books.json` gives
  `textFileURL` (no nikud) + `nikudMetegFileURL` (with). **CC BY-SA 4.0**. Texts live behind mutable
  library.dicta.org.il URLs → snapshot + hash at ingest. Auto-scanned → OCR/nikud errors; verify vs
  Sefaria where titles overlap.

## MEDIUM value

- **Dicta-LM / DictaBERT + Nakdan** — Hebrew NLP models (Apache 2.0) + diacritization test corpora
  (public domain). Not a corpus; the way to *add nikud* to unpointed text. Pin by HF revision.
- **Open Siddur Project** — github.com/opensiddur. Best dedicated open **liturgy** corpus (siddurim,
  haggadot, …), TEI XML. Licensing is **per-work** (CC0 / CC-BY / CC-BY-SA 3.0) — repos have **no
  repo-level LICENSE**; read each work's TEI header. Data is somewhat staging-stage.
- **PocketTorah** — github.com/rneiss/PocketTorah + archive.org. Professionally chanted Torah/Haftarah
  **audio** + JPS 1917 text, JSON metadata. Creators announced **CC BY-SA** for audio, **but the live
  repo has no LICENSE** — confirm before redistributing. Adds an audio modality none of the three have.
- **Hebrew Wikisource** — MediaWiki XML **dated dumps** (immutable pin) or live API. **CC BY-SA** +
  PD-old. Sefaria already imported a Wikisource Tanach → core texts duplicative; value is the long tail.
  Parsing-heavy wikitext.
- **otzaria-library (Sivan22)** — pre-assembled aggregation of Sefaria/Orayta/Dicta/Wikisource/etc.,
  `otzaria_latest.zip` (~1.2 GB). **No LICENSE on the data** (app is Unlicense; data is an amalgam of
  differently-licensed upstreams) → don't redistribute the bundle. **Real value = its conversion
  scripts** (`*ToOtzaria`) as a reference for normalizing these upstreams. Go to upstreams directly.
- **Mechon Mamre** — Tanakh (Breuer, vocalized), Mishnah/Tosefta/Yerushalmi/Bavli, **Mishneh Torah**
  (Yemenite-MS-based, excellent). Bulk download. License: *"free to use, **subject to their
  copyright**"* → **NOT open**; do not assume redistribution rights. Flagged not-clearly-free.

## LOW / situational

- **AlHaTorah.org** — superb Mikraot Gedolot with manuscript-based critical editions not in
  Sefaria/Orayta, but **website only, no bulk export, no clear license** (scholars' tier paywalled).
  Reference, not ingest.
- **CAL (Aramaic Lexicon) / Targumim** — authoritative, but **not bulk-downloadable / academic-use**;
  the free downloadable piece is just PD Etheridge English. Use Sefaria's existing Targumim instead.
- **Aleppo/Leningrad Codex imaging** — manuscript **page scans / PDF**, not machine-readable; Aleppo
  images carry attribution + **non-commercial** terms. Only for facsimile display / codex-variant work.
- **HebrewBooks.org** — huge, but **PDF scans** (poor OCR, mixed licensing). Avoid for structured ingest.

## Pinning summary

- **Immutable identifier (prefer):** tanach.us (official SHA-256 + build #); STEPBible, HebrewLexicon,
  Dicta Talmud, Open Siddur (git commit SHA); Hebrew Wikisource (dated dump); Dicta models (HF revision).
- **Mutable → snapshot + hash at ingest:** Dicta Library texts (library.dicta.org.il URLs);
  otzaria-library (`latest` tag is re-published — pin repo SHA + hash the zip); Mechon Mamre / AlHaTorah
  pages.

## Licensing watch-outs (be skeptical)

- **Mechon Mamre:** "subject to their copyright" = NOT open.
- **otzaria-library:** no LICENSE on data; amalgam of upstreams — use upstreams directly.
- **PocketTorah:** CC BY-SA per announcement, but no in-repo LICENSE — confirm before redistributing audio.
- **STEPBible:** take TAHOT/TAGNT/lexicons (CC BY 4.0); skip `TTESV` (CC BY-NC).
- **AlHaTorah / CAL:** unique content, no open bulk license — reference, not ingest.
- **tanach.us fonts** are separately licensed (text is free, fonts are not).

## One-line recommendation

Add **STEPBible TAHOT** (morphology/lemmas, CC BY), **Dicta Talmud-with-Nikud** (vocalized Bavli, CC
BY-SA), **OpenScriptures HebrewLexicon** (BDB+Strong's, CC BY), and **tanach.us** (cantillation +
SHA-pinning). Layer **Dicta Nakdan** to vocalize unpointed text, **Open Siddur** for liturgy, and
**PocketTorah** for audio.
