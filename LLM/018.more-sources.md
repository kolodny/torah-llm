# 018 — More Jewish-origin sources (commentaries, Talmud, responsa, Biur)

**Date:** 2026-06-09

Worked through the remaining Jewish-origin sources from [014](014.data-sources.md), researched in
parallel by four subagents (Dicta Talmud, Dicta Library, PocketTorah, Hebrew Wikisource). Now **6
adapters**; 34 content-bearing books, 58 editions, ~21k links.

## Orayta commentaries — Ramban / Ibn Ezra / Sforno on the Torah
Extended `orayta.ts` from Rashi-only to all four (15 commentary files under `04_rmbn` / `05_abn_uzra`
/ `06_sporno`, same `! {verse}` + `<b>`-lemma format). Fix: the comment split is now case-insensitive
(Ramban uses `<B>`). Each becomes the Orayta edition of the existing canonical "X on <Book>" node
(**dedup by title**). **Commentary→base links** are derived in the adapter (`Genesis 1:1 ↔ Ramban on
Genesis 1:1:1`) using the Sefaria adapter's canonical `sortLink` ordering, so the Rashi links
**dedupe** with Sefaria's. Genesis 1:1 now links to Rashi + Ramban + Ibn Ezra + Sforno.

## Dicta Talmud with Nikud — `dicta-talmud.ts`
Vocalized Vilna Talmud (CC BY-SA; Dicta). `*** דף N ***` markers ('.'=amud a, ':'=b) → `Daf:Line`
refs matching Sefaria's Talmud addressing; standalone `מתני'`/`גמ'` markers skipped. Tractate titles =
Sefaria's (repo dir name). No links in the data. Subset: **Berakhot, Megillah** (ktiv haser).

## Dicta Library — `dicta-library.ts`
OCR'd rabbinic texts (CC BY-SA; Dicta). Per-book ZIP of per-page `.txt` (downloaded from the mutable
`files.dicta.org.il`, unzipped via `unzip`). `סימן N` headings + `. `-paragraphs → `siman:paragraph`.
Mostly obscure Acharonim absent from Sefaria → **new canonical books** under a new top-level
**`Dicta Library`** category (first adapter to mint a category). No structured links (citations are
free-text prose). Subset: **Achiezer, Levush Mordechai on Gittin**.

## Hebrew Wikisource — the Biur — `hewikisource.ts`
~90% of he.wikisource duplicates Sefaria; the unique slice is the **Biur** (Wikisource's own
modern-Hebrew Tanakh commentary, CC BY-SA 4.0). Fetched via the MediaWiki API (whitelisted pages).
`{{ביאור:אות-פסוק|book|ch|v}}` anchors give refs; `{{ב|word|gloss}}` → `word (gloss)`. Modeled as a
**verse-aligned edition** of the base book, so it dedupes into canonical Genesis/Psalms/Deuteronomy
(Genesis now has 6 editions). Subset: **Biur on Genesis (1–3), Psalms (1, 23), Deuteronomy (6)**.

## PocketTorah — skipped
Its text is ~100% duplicative (same WLC Hebrew + JPS 1917 we already ingest); its only real value is
chanting **audio** + per-word timing, which the text-only schema can't hold. Documented for later
(would need a sibling `audio` table linking to the archive.org files; CC BY-SA).

## build-master — global link prune
Added a post-ingest pass: `DELETE FROM links` where either endpoint has no content row. This
generalizes the old Sefaria-only "both endpoints in the master DB" rule across all sources, so
adapters can derive links liberally and only resolvable ones survive.

## Verified e2e (Playwright)
Berakhot loads (1114 rows / 125 daf, vocalized); Achiezer loads (895 rows / 13 simanim, OCR); Genesis
shows the Biur edition + verse commentary links to Rashi/Ramban/Ibn Ezra/Sforno; all 34 books appear
in the catalog. `BOOT_VERSION` 8 → 9 (Orayta commentaries + Dicta Talmud) → 10 (Dicta Library + Biur).

**UI note:** the catalog renders a flat list of content-bearing books; category headers (Tanakh,
Talmud, Liturgy, Dicta Library) aren't shown as group labels yet — a grouping improvement for later.
